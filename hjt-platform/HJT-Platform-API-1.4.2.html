    会捷通平台自服务管理API 说明

    V1.4.2

    一、常量

    会议的分屏模式

    分屏模式的值列举如下： auto 1X1，1X2，2X1，2X1H，1X2V， 1TP2H，1BP2H，1LP2V， 2X2，1BP3H，1LP3V，1TP3H， 1LP4V，1BP4H，1TP4H， 1P5，2X3， 1P7， 3X3，1MP8，1BP8，1TP8， 2MP8，1CP12，4X4，1P16，1CP16，4X5，4X5S， 1P20，1CP20，5X5，5X6，5X6S，6X6

    说明： T表示Top，B表示Bottom，C表示Center，M表示Middle，L表示Left，H表示Horizontal，V表示Vertical，P表示Plus，S表示Square

    会议状态

    取值（类型为String） 说明
    APPROVED 预约成功
    ONGOING 正在进行中
    FINISHED 已结束
    FAILED 失败
    DELETED 已被删除

    用户状态

    取值（类型为int） 说明
    0 激活
    1 锁定
    2 停用

    用户角色

    取值（类型为String） 说明
    SYS_ADMIN 系统管理员
    AGENT_USER 客服代表
    ORG_ADMIN 公司管理员
    DEPT_ADMIN 部门管理员
    USER 普通用户

    终端状态

    取值（类型为String） 说明
    OFFLINE 离线
    ONLINE 在线
    INCALL 呼叫中
    UNKNOWN 未知

    终端类型

    取值（类型为String） 说明
    M16 -
    M18 -
    VE/VL -
    HEXMEET 软终端
    OTHER 其他类型

    呼叫协议

    取值（类型为String） 说明
    SVC -
    SIP -
    H.323 -

    双流分屏

    取值（类型为String） 说明
    oneByOne 平衡
    contentOnly 仅双流
    largeContentSmallPeople 双流优先
    largePeopleSmallContent people流优先
    peopleOnly 仅people流

    录制/直播类型

    取值（类型为String） 说明
    GW_WEBCAST 高清
    GW_FHDWEBCAST 超清

    录制状态

    取值（类型为String） 说明
    started 录制中
    paused 已暂停录制
    resumed 已恢复录制
    stopped 已停止录制

    直播状态

    取值（类型为String） 说明
    started 直播中
    stopped 已停止直播

    文件类型

    取值（类型为String） 说明
    AVATAR 用户头像
    ORG_LOGO 公司LOGO
    CONF_RECORD 会议的录制文件
    WHITEBOARD_FILE 会议的白板文件

    文件链接的用途

    取值（类型为String） 说明
    PLAYBACK 播放
    DOWNLOAD 下载
    COPY_URL 复制链接

    会议室的容量

    即会议室能同时容许多少终端入会，容量值列举如下：

    10, 25, 50, 100, 200, 400, 1000.

    二、消息结构体定义

    RestWebLoginReq

    简要描述：

    • 自服务管理和平台资源管理的登录请求体

    参数说明:

    字段 类型 必选 默认 说明
    account String 是 用户名
    password String 是 密码,SHA-1加密
    intranet String 是 若为内网访问，则值为"true";若为外网访问，则值为"false".

    示例：

     { 
     "account" : "caopi", 
     "password" : "7c4a8d09ca3762af61e59520943dc26494f8941b",
	 "intranet" : "true";
 }

    RestWebLoginResp

    简要描述：

    • 自服务管理和平台资源管理的登录返回

    参数说明:

    字段 类型 说明 备注
    token String 登录令牌 -
    profile RestUser 用户信息 -
    zoneRole String 可用区的类型 可能的值有：Standard_Zone，Center_Zone，Standalone_Zone
    reportUrlPrefix String 统计报告服务地址的前缀 protocol+host+port
    fileDownloadUrlPrefix String 文件下载服务地址的前缀 protocol+host+port
    portModeEnabled boolean 是否为端口模式 -

    RestConfReq

    简要描述：

    • 预约会议请求体

    参数说明:

    字段 类型 必选 默认 说明
    name String 是 - 会议名称,长度不可超过128个字节
    startTime long 是 - 会议的开始时刻（以毫秒为单位的Unix时间戳），0表示该会议立即开始。
    duration long 是 - 会议的持续时长（单位为毫秒）,建议值为3600000，即一个小时
    lecturerEpId long 否 - 被设置为主会场的某个终端的id
    lecturerUserId long 否 - 被设置为主会场的某个用户的id
    layout String 是 - 取值见常量：会议的分屏模式
    profile String 是 - 会议质量，"SVC"表示质量优先，"SVC_ECO"表示带宽优先，建议值为"SVC".
    password String 否 - 会议密码
    description String 否 - 会议备注
    endpointIds List(Long) 否 - 与会终端id的列表
    userIds List(Long) 否 - 与会用户id的列表
    roomId long 否 - 指定的云会议室的id（适用于云会议室模式）
    maxPartyCount int 是 - 最大入会方数
    useRandomNumericId boolean 是 false 使用随机号码作为会议号码，端口模式下必须设为true
    redialingEnabled boolean 否 false 自动重拨
    recordingEnabled boolean 否 false 录制
    recordingProfile String 否 - 取值见常量：录制/直播类型,建议值为"GW_WEBCAST".
    recordingLayout String 是 - 取值见常量：双流分屏，建议值为"oneByOne".
    liveStreamingUrl String 否 - 推流地址
    adminUserId long 否 - 备选会控者的id
    subtitle RestSubtitle 是 - 字幕
    autoMutingEnabled boolean 否 false 开启自动静音
    allowAnonymousParty boolean 否 false 允许匿名用户入会

    RestSubtitle

    简要描述：

    • 会议字幕

    参数说明:

    字段 类型 说明 必选 备注
    on boolean 启用字幕 是 -
    content String 字幕内容 是 -
    displaySpeed int 滚动速度 是 建议取值：0表示静止，2表示慢速，5表示中速，8表示快速。
    displayRepetitions int 滚动次数 是 滚动速度大于0时才生效.最小值为1,最大值为20,建议值为20.
    verticalBorder int 垂直位置 是 最小值为0，最大值为100，建议值为50.
    transparency int 透明度 是 最小值为0，最大值为100，建议值为50.
    fontSize int 字体大小 是 最小值为9，最大值为32，建议值为24.
    foregroundColor String 前景色 是 十六进制颜色码，建议值为"#FFFF00".
    backgroundColor String 背景色 是 十六进制颜色码，建议值为"#3300CC".

    RestEndpointReq

    简要描述：

    • 添加/修改终端的请求体

    参数说明:

    字段 类型 必选 默认 说明 备注
    id long 否 - 终端的ID 修改终端时必须赋值
    systemName String 是 - 名称 -
    ip String 否 - IP地址 -
    callProtocol String 是 - 呼叫协议 取值见常量：呼叫协议
    sipUrl String 否 - SIP号码 callProtocol的值为“SIP”时，需要为本字段和ip字段中的至少一项赋值
    description String 否 - 对终端的描述 -
    deviceSn String 否 - 终端的序列号 callProtocol的值为“SVC”时，必须为本字段赋值

    deptId long 否 - 终端所属部门的id -
    zoneName String 否 - 终端所属可用区的名称 callProtocol的值为“SIP”或“H.323”时，必须为本字段赋值
