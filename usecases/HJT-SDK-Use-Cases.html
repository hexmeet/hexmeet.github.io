    会捷通视频通信客户端 SDK 应用场景示例

    会捷通视频通信客户端 SDK 是⼀款视频通信软终端开发套件，开发者可以利⽤此 SDK 开发出具有清晰流畅的⾳视频体验和⾼清内容协作的⾳视频会议终端应用。

    会捷通视频通信客户端 SDK 具备强⼤的⽹络适应性，和独特的⾳视频抗⽹络丢包算法，配合会捷通云视讯平台使⽤，可以保证在 30% ⽹络丢包环境下视频依然清晰流畅，即使⽹络丢包⾼达 50%，依然可以保证⾳频通畅。 会捷通客户端 SDK 提供了丰富，⽽且简单易⽤的 API 接⼝。开发者不需要掌握丰富的⾳视频和信令相关知识，也可以使⽤本 SDK 开发出专业的视频会议软终端应用。

    利用会捷通 SDK，开发者可以发挥自己的聪明才智，将其应用在各种各样的场景之中，使各类行业应用插上视频的翅膀，实现云 + 视频、业务 + 视频的理想解决方案。

    视频会议

    视频会议解决方案是会捷通 SDK 的主要应用场景，利用会捷通 SDK，开发者可以开发出具有清晰流畅的⾳视频体验和⾼清内容协作的专业⾳视频会议终端应用。会捷通 SDK 客户端与会捷通平台相结合，打造完美视频会议方案。

    智慧党建

    习近平总书记深刻指出，将信息技术应用到党建工作中，既是信息化时代发展的客观要求，也是党建工作改革创新的必然要求。随着现代科学技术的迅猛发展，信息化已成为人类社会发展的必然趋势。各级地方将信息技术运用于党员教育培训，开发了在线学习平台、微博微信手机 APP 等即时学习工具，丰富了党员教育方式、拓宽了党员教育途径。通过视频会议或通话的形式进行的在线学习，是其中不可或缺的一种形式，利用这种形式可以提高组织学习的效率和效果，同时可以有效地节约学习费用，已经被越来越广泛的应用于实际的党建学习工作之中。运用会捷通 SDK 可以将视频通话能力加入到各种形式的学习 App， 甚至是视频终端设备之中，有效助力开发者打造功能丰富的智慧党建解决方案。
